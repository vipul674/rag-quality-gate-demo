# Resume Guidelines

## Resume Requirement

Every applicant must submit a resume as part of the NovaTech Summer Internship Program 2026 application.

The resume is used during application screening, interview preparation, and final selection review.

## Accepted Format

The resume must be submitted as a **PDF file**.

## File Naming Rule

Use the following file naming format:

`FirstName_LastName_Track_Resume.pdf`

Example:

`Aarav_Sharma_AIEngineering_Resume.pdf`

## Resume Length

The resume must be:

- **One page** for undergraduate students.
- Up to **two pages** only for postgraduate students with significant research or work experience.

## Required Resume Sections

The resume must include:

1. Full name
2. Email address
3. Phone number
4. LinkedIn profile, if available
5. GitHub or portfolio link, if relevant
6. Education details
7. Technical skills
8. Projects
9. Internship or work experience, if any
10. Certifications or achievements, if relevant

## Project Section Rules

Each project should include:

- Project name
- Problem solved
- Technologies used
- Applicant's personal contribution
- Measurable result, if available
- GitHub, demo, paper, or portfolio link, if available

Weak project description:

> Built a chatbot using AI.

Strong project description:

> Built a document QA chatbot using Python, FastAPI, ChromaDB, and OpenAI embeddings. Implemented PDF ingestion, semantic search, and answer citation. Reduced manual policy search time by 60% in testing.

## Skills Section Rules

Applicants should list skills they can explain in an interview.

Do not list tools, frameworks, or programming languages that the applicant has never used.

## ATS and Screening Rules

The resume should use:

- Clear section headings
- Standard fonts
- Simple formatting
- No tables that break text extraction
- No decorative icons
- No images of text
- No excessive colors

## Prohibited Resume Content

The resume must not include:

- False experience
- Fake internships
- Misleading project ownership
- Private credentials
- Aadhaar number, passport number, or sensitive identity numbers
- Irrelevant personal details
- Large blocks of copied text from online templates

## Resume Review Criteria

Recruiters evaluate resumes based on:

1. Eligibility match
2. Relevant skills
3. Project quality
4. Technical depth
5. Clarity of communication
6. Evidence of execution
7. Alignment with selected internship track

## Common Resume Mistakes

Common mistakes include:

- Submitting a resume longer than allowed
- Forgetting project links
- Listing too many unrelated skills
- Using vague project descriptions
- Using an editable document instead of PDF
- Missing graduation year
- Using an unprofessional email address
