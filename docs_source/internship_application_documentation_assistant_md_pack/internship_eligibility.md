# Internship Eligibility

## Program Name

NovaTech Summer Internship Program 2026

## Eligible Candidates

Applicants must meet all of the following eligibility requirements:

1. Must be currently enrolled in a recognized undergraduate or postgraduate program.
2. Must graduate in **2026, 2027, or 2028**.
3. Must be available for the full internship period from **June 1, 2026 to August 7, 2026**.
4. Must be legally eligible to complete a paid internship in their country of residence.
5. Must have access to a laptop and stable internet connection.
6. Must be able to commit at least **35 hours per week** during the internship.

## Eligible Academic Backgrounds

Students from the following fields are eligible:

- Computer Science
- Information Technology
- Artificial Intelligence
- Data Science
- Electronics and Communication
- Cybersecurity
- Mathematics or Statistics
- Product Design
- Business Analytics

Students from other disciplines may apply if they can demonstrate relevant skills through projects, coursework, internships, certifications, or open-source contributions.

## Minimum Academic Requirement

Applicants must have one of the following:

- Minimum **7.0 CGPA out of 10**, or
- Minimum **70% aggregate**, or
- Equivalent academic standing based on their institution's grading system.

## Skills Expected by Track

### Software Engineering Track

- Programming fundamentals
- Data structures and algorithms
- Git and GitHub
- Basic web development or backend development

### AI Engineering Track

- Python
- Machine learning fundamentals
- Prompt engineering basics
- APIs
- Basic understanding of embeddings or vector databases

### Data Science Track

- Python or R
- SQL
- Statistics
- Data cleaning and visualization

### Cybersecurity Track

- Networking fundamentals
- Linux basics
- Web security fundamentals
- Responsible disclosure awareness

### Product Design Track

- UX research basics
- Wireframing
- Figma or similar design tools
- Portfolio with at least one case study

## Ineligible Candidates

Applicants are not eligible if they:

- Have already graduated before 2026.
- Cannot commit to the full internship timeline.
- Submit false academic, identity, or project information.
- Are unable to provide required documents before the deadline.
- Have previously been removed from a NovaTech program for misconduct.

## Required Proof of Eligibility

Shortlisted applicants may be asked to provide:

- College ID card
- Recent transcript or marksheet
- Government-issued identification
- Bonafide certificate or enrollment letter
- Graduation year confirmation
