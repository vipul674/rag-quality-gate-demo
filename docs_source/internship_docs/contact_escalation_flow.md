# Contact and Escalation Flow

## Purpose

This document explains how applicants should contact NovaTech for support during the Summer Internship Program 2026 application process.

## General Support Rule

Applicants should use the same email address that they used in the application form.

Every support email should include:

- Full name
- Application ID, if available
- Internship track
- Registered email address
- Clear description of the issue
- Screenshot, if relevant

## Contact Channels

| Issue Type | Contact Email | Expected Response Time |
|---|---|---:|
| General application support | internships-support@novatech.example | 3 business days |
| Resume upload issue | internships-support@novatech.example | 3 business days |
| Online assessment issue | assessment-help@novatech.example | 24 hours during assessment window |
| Interview scheduling issue | interview-coordination@novatech.example | 2 business days |
| Offer letter issue | offers@novatech.example | 3 business days |
| Accessibility accommodation | accommodations@novatech.example | 5 business days |
| Urgent security concern | security@novatech.example | 24 hours |

## Escalation Flow

Applicants should follow this escalation order:

1. Contact the relevant support email.
2. Wait for the expected response time.
3. Reply to the same email thread with additional details if needed.
4. If there is no response after the expected time, escalate to `internships-escalation@novatech.example`.
5. For time-sensitive interview or assessment issues, include the word `URGENT` in the subject line only if action is needed within 24 hours.

## Subject Line Format

Recommended subject format:

`[NovaTech Internship 2026] Issue Type - Full Name - Application ID`

Example:

`[NovaTech Internship 2026] Resume Upload Issue - Aarav Sharma - NT2026-10482`

## What Not to Do

Applicants should not:

- Send the same issue to all email addresses.
- Use social media comments for private application issues.
- Send personal identity documents unless requested.
- Share passwords or OTPs.
- Create multiple applications to fix a mistake.
- Threaten or harass support staff.

## Emergency Issues

Emergency issues include:

- Assessment platform failure during the active test window.
- Interview link not working within 30 minutes of interview time.
- Incorrect offer acceptance deadline.
- Confirmed account compromise or suspicious email claiming to be from NovaTech.

Emergency issues should be sent to the relevant channel with `URGENT` in the subject line.

## Accessibility Support

Applicants needing accommodations should contact `accommodations@novatech.example`.

Examples of accommodations include:

- Extended assessment time
- Screen reader-compatible material
- Alternative interview format
- Captioning support
- Flexible scheduling for documented needs

Accommodation requests should be submitted as early as possible, preferably before **March 10, 2026**.
