# Application Process

## Overview

The NovaTech Summer Internship Program 2026 application process has five stages:

1. Online application submission
2. Document screening
3. Online assessment
4. Technical or portfolio interview
5. Final selection and offer confirmation

## Stage 1: Online Application Submission

Applicants must complete the online application form before the deadline.

Required form fields include:

- Full name
- Email address
- Phone number
- College or university name
- Degree and branch
- Graduation year
- Internship track preference
- Resume upload
- Project link or portfolio link
- Short answer responses

## Stage 2: Document Screening

The recruiting team verifies:

- Eligibility
- Graduation year
- Resume completeness
- Project relevance
- Academic requirement
- Required document availability

Applicants with incomplete or inconsistent documentation may be rejected at this stage.

## Stage 3: Online Assessment

Most technical tracks require an online assessment.

Assessment format depends on track:

| Track | Assessment Type |
|---|---|
| Software Engineering | Coding and debugging |
| AI Engineering | Python, ML basics, prompt design, API reasoning |
| Data Science | SQL, statistics, data interpretation |
| Cybersecurity | Web security basics, Linux, networking |
| Product Design | Design critique and problem-solving task |

The assessment window is **March 22, 2026 to March 24, 2026**.

## Stage 4: Interview

Shortlisted applicants attend one or two interviews.

Possible interview types:

- Technical interview
- Project deep-dive
- Portfolio review
- Problem-solving discussion
- Behavioral interview

Applicants should be ready to explain every project, skill, and claim mentioned in their resume.

## Stage 5: Final Selection

Final decisions are based on:

1. Eligibility
2. Resume quality
3. Assessment performance
4. Interview performance
5. Project depth
6. Communication skills
7. Track fit

## Offer Confirmation

Selected applicants receive an offer by email.

Applicants must confirm acceptance within **5 calendar days** of receiving the offer.

Failure to confirm within the deadline may result in the offer being reassigned to a waitlisted candidate.

## Required Documents Before Joining

Selected applicants must submit:

- Signed offer letter
- Government-issued ID
- College ID card
- Recent transcript or marksheet
- Bank details for stipend processing
- Emergency contact information
- Confidentiality agreement
- Internship participation consent form, if required by the college
