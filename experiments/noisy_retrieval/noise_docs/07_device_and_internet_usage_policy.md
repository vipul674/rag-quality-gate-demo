# Device and Internet Usage Policy

**Company:** Apex Nimbus Labs  
**Effective Date:** January 1, 2026  
**Policy Owner:** IT and Security Team

## 1. Purpose

This policy sets expectations for responsible use of company devices, networks, and internet access.

## 2. Company Devices

Employees must:

- Keep devices secure.
- Install updates promptly.
- Use approved security software.
- Report lost or stolen devices immediately.
- Return devices when employment ends.

## 3. Personal Use

Limited personal use may be allowed if it does not interfere with work, violate policy, or create risk.

## 4. Prohibited Activity

Employees must not use company systems to:

- Access illegal content.
- Harass or threaten others.
- Download unauthorized software.
- Store personal sensitive files.
- Mine cryptocurrency.
- Disable security controls.
- Share confidential data without authorization.

## 5. Monitoring

Apex Nimbus Labs may monitor company-owned systems and networks for security, compliance, and operational purposes where permitted by law.

## 6. Software Installation

Only approved software may be installed on company devices.

## 7. Lost or Stolen Devices

Employees must report loss or theft immediately to IT and Security.
